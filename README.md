# Diabetes-onset-prediction-

## 📌 Overview
This project is a Machine Learning-based web application that predicts whether a person is diabetic or not based on medical parameters. It uses the **XGBoost algorithm** for accurate prediction and is deployed using a **Flask web application**.

---

## 🎯 Objective
- Predict diabetes at an early stage using machine learning  
- Help in faster and more reliable diagnosis  
- Reduce manual effort in medical testing  
- Provide a simple web-based prediction system  

---

## 🧠 Machine Learning Model
- Algorithm Used: XGBoost Classifier  
- Focus: High accuracy and better recall  
- Feature Engineering: BMI category, glucose level risk, etc.  
- Model trained on standard medical dataset  

---

## 📊 Features Used
- Glucose Level  
- BMI (Body Mass Index)  
- Age  
- Blood Pressure  
- Insulin  
- Skin Thickness  
- Number of Pregnancies  

---

## 🏗️ Project Workflow
User Input → HTML Form → Flask Backend → ML Model (XGBoost) → Prediction → Result Display  

---

## 💻 Technologies Used
- Python  
- Flask  
- XGBoost  
- HTML, CSS, JavaScript  
- Pandas, NumPy  
- Matplotlib, Seaborn  

---

## 📈 Results
- High accuracy achieved using XGBoost  
- Improved recall for medical safety  
- Real-time prediction system  
- Simple and interactive web interface  
- Visual output for better understanding
